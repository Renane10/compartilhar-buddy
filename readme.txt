Compartilhar Buddy é um plugin WordPress que permite compartilhar atividades do BuddyPress com usuários não logados.

== Descrição ==

Compartilhar Buddy é um plugin WordPress que adiciona funcionalidade de compartilhamento às atividades do BuddyPress. Os usuários logados podem compartilhar uma atividade específica e gerar uma página de compartilhamento para que usuários não logados possam visualizar a atividade compartilhada.

Recursos do plugin:

Adiciona um botão de compartilhamento às atividades do BuddyPress para usuários logados.
Gera uma página de compartilhamento que exibe a atividade compartilhada para usuários não logados.
Utiliza a biblioteca Toastr.js para exibir mensagens de sucesso e erro ao copiar a URL da atividade.
Suporta a cópia da URL da atividade para a área de transferência do usuário.
Compatível com o plugin Elementor para personalização visual da página de compartilhamento.
== Instalação ==

Faça o upload do arquivo compartilhar-buddy para o diretório /wp-content/plugins/.
Ative o plugin por meio do menu 'Plugins' no WordPress.
Utilize o shortcode [compartilhar_buddy_shortcode] em qualquer página ou post para exibir o botão de compartilhamento.
== Perguntas Frequentes ==

Como adicionar o botão de compartilhamento nas atividades do BuddyPress?
R: Após ativar o plugin, o botão de compartilhamento será automaticamente adicionado às atividades do BuddyPress para usuários logados.

Como personalizar a página de compartilhamento?
R: A página de compartilhamento pode ser personalizada utilizando o plugin Elementor. Basta criar uma página e adicionar o shortcode [compartilhar_buddy_shortcode] para exibir o conteúdo compartilhado.
